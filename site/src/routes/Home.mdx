## Quick Start

```bash
npm install use-kbd  # or: pnpm add use-kbd
```

<QuickStartCode />

Basic steps:

1. **Drop-in UI components**: <ShortcutsModalLink />, <OmnibarLink />, <LookupModalLink />, <SequenceModalLink />, <ActionLinkLink />
2. **Register functions as "actions"** with <UseActionLink />
3. **Easy theming** with CSS variables

## Features
- **Editable Shortcuts** - <KbdModal />, edit shortcuts in modal to rebind
- **Global Omnibar** - <KbdOmnibar /> Search all actions across pages
- **Sequences** - Multi-key shortcuts like <Kbd action="link:/table" tooltip="Navigate to Table demo" />
- **Persisted Bindings** - Custom shortcuts saved to localStorage
